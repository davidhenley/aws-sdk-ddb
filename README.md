AWS SDK with DynamoDB
=====================

Usage
-----

Run the following in sequence to ensure database calls work.

**Install**

`npm install`

**Add Amazon AWS Keys**

Modify `src/config/aws-keys.json` using example file.

**Create A Table**

`npm run create`

**List Tables**

`npm run list`

**Describe Table**

`npm run describe`

**Delete Table**

`npm run delete`

**Put Item**

`npm run put`

**Get Item**

`npm run get`
